**一、项目背景**
一方面：公司业务是反爬虫的，动手实现爬虫有助于了解爬虫的原理，知己知彼方能取胜
另一方面：练习golang的基础语言，channel，协程，ElasticSearch,redis,mysql等基础开源组件的使用

本项目麻雀虽小，但是五脏俱全。
